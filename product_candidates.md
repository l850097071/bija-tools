## 候选1: NC级学术写作深度润色 🥇

### 产品定义
不卖语法润色，卖**NC级逻辑链打磨**——帮作者把论文从"结果堆砌"变成"有说服力的科学故事"。

### 为什么是它
- **市场验证**: NC版面费¥52,000/篇 → 作者已投入巨额成本，润色费敏感度低
- **竞争空白**: 10家竞品全做语法润色，无一做逻辑链重构
- **宿主优势**: 真实NC发表经验 = 不可替代的信任背书
- **价格修正** (2026-05-31被动信号): 中国市场实际支付¥800-1,700 (AJE/小木虫)，Nature金牌国际价$587-1,056与国内市场脱节
- **修正定价**: 深度编辑¥2,500-3,500 (AJE价的2-3x, 差异化溢价) / NC冲刺¥5,000 (<版面费¥52,000的10%)

### 三维评分
| 维度 | 分数 | 理由 |
|------|------|------|
| 可行性 | 3 | ⚠️ 修正: 3/3访谈+论坛数据→中国学术圈付费意愿弱, 需验证"PI课题经费报销"渠道 |
| 收入潜力 | 3 | ⚠️ 修正: ¥2,500-5,000/篇 × 保守10篇/年 = ¥25,000-50,000/年 (原高估5-10x) |
| 科研协同 | 4 | 宿主写NC论文的过程=服务能力积累。润色他人论文加深科学判断力 |
| **总分** | **36/125** | (修正后, 原64→36) |

### 服务内容
1. 论证链重构（不是改语法，是改"这个实验为什么放在这里"）
2. 审稿人视角预判（"这里审稿人会问什么？"）
3. 讨论部分深化（与现有文献对话，而非罗列结果）
4. 学术英语润色（基础层，AI+人工）

### 最小验证
- [ ] 找3位NC/AM目标作者做15分钟痛点访谈
- [ ] 免费润色1篇→收集反馈→迭代服务定义
- [ ] 第一位付费用户 (验证定价)

### 风险
- 学术伦理边界（润色≠代写——需要明确的伦理红线声明）
- 人工成本高，难以规模化
- 依赖宿主个人品牌

## 候选2: 催化DFT自由能计算CRO 🥈

### 产品定义
提供VASP自由能计算服务——工业界客户提供结构，我们返回自由能谱+催化机理分析。

### 为什么是它
- **宿主核心能力**: CHE自由能计算pipeline + ~630条DFT数据
- **工业需求**: 制药/化工企业需要DFT计算但不想自建团队
- **竞争少**: 计算化学CRO无成熟线上市场——先发优势

### 三维评分
| 维度 | 分数 | 理由 |
|------|------|------|
| 可行性 | 3 | HPC资源已有，pipeline就绪。但获客需线下关系(企业不搜索"DFT外包") |
| 收入潜力 | 3 | 工业客户单项目¥10,000-50,000，但获客周期长 |
| 科研协同 | 5 | 直接复用宿主计算资源+方法论，100%协同 |
| **总分** | **45/125** | |

### 服务内容
1. 自由能计算 (VASP + 热力学修正)
2. 催化机理分析 (过渡态+反应路径)
3. 计算报告 (工业级交付物，非学术论文格式)

### 最小验证
- [ ] 找2位工业界计算化学家访谈: 外包需求？价格预期？
- [ ] 完成1个付费试算项目
- [ ] 确定获客渠道（学术会议？LinkedIn？行业论坛？）

### 风险
- 获客严重依赖宿主个人网络
- 计算质量争议时责任边界模糊
- 企业客户保密协议复杂

## 候选3: Bija自进化Agent框架 (开源+托管) 🥉

### 产品定义
开源Agent框架 + 托管服务 + 企业版。差异化卖点: "会自我进化的Agent——不只会执行任务，还会发现自己缺什么→主动改进"。

### 为什么是它
- **市场大**: Agent市场2026年1830亿→2033年1.2万亿
- **差异化明确**: DSEIM自进化循环是独特的——其他Agent框架不会自我改进
- **一人公司模式**: 2026年已验证"一人+AI=20人团队产出"

### 三维评分
| 维度 | 分数 | 理由 |
|------|------|------|
| 可行性 | 3 | 技术基础已有(Bija方法论→产品化)。但开源项目冷启动难 |
| 收入潜力 | 4 | Agent SaaS可达$10M+ ARR。但需要时间+社区积累 |
| 科研协同 | 2 | 与宿主科研方向无关，纯粹的外部化 |
| **总分** | **24/125** | |

### 服务内容
1. 开源核心框架 (GitHub)
2. 托管服务 ($99/月)
3. 企业版 ($5000/年起——私有部署+定制)
4. 培训/咨询

### 最小验证
- [ ] 找3位AI独立开发者访谈: 自进化功能是否刚需？
- [ ] GitHub MVP上线→观察star增长
- [ ] 10个活跃用户后→开始收费

### 风险
- 竞品多(Velobase/OpenClaw/CROO)，同质化严重
- 与宿主科研方向脱节——可能分散注意力
- 开源社区运营成本高
