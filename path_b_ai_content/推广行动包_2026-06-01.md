## 渠道2: Reddit r/comp_chem

**Title**: [Free Tool] bija-tools v0.1 — CLI toolkit for VASP calculations (input gen → pre-submit validation → OUTCAR parsing → lit search)

Just shipped a zero-dep Python CLI for computational chemists doing DFT:

- `vaspgen`: CIF/formula → INCAR+POSCAR+KPOINTS+job script (59 elements, auto-detects metal/magnetism/periodicity)
- `gate`: 10-gate pre-submit validation for VASP/PWmat/Gaussian — catches POTCAR mismatches, missing dispersion corrections, insufficient K-points BEFORE you burn GPU hours
- `outcarp`: OUTCAR → human-readable convergence report with energies, forces, timing
- `litsearch`: Semantic Scholar API → structured literature review
- `pipeline`: One command end-to-end: `bija-tools pipeline "CO2 reduction" --formula Co3O4`

Also includes 8 calculation recipes (CO2 adsorption, NEB, phonon, VASPsol, band structure, Bader, bulk opt, DFT+U) with INCAR params + known pitfalls, searchable in a browser-based UI.

```bash
pip install git+https://github.com/l850097071/bija-tools.git
```

MIT license. Free forever. Would love feedback from the community.

**Subject**: [ANN] bija-tools v0.1 — open-source CLI toolkit for VASP workflow automation

Dear VASP community,

I'd like to share bija-tools, a free open-source command-line toolkit designed to reduce the operational overhead of VASP calculations:

- Automated input file generation from CIF/formula
- 10-gate pre-submit validation (element matching, K-point density, dispersion corrections, etc.)
- OUTCAR parsing with structured convergence reports
- Semantic Scholar literature search integration
- Browser-based recipe database for common calculation types

Install: `pip install git+https://github.com/l850097071/bija-tools.git`
Docs: https://github.com/l850097071/bija-tools

The toolkit is MIT-licensed and supports VASP, PWmat, and Gaussian. Feedback and contributions welcome.
