## 错误1: SCF不收敛——还在硬扛默认参数

**常见死法**: SCF跑了60步电子步还没收敛，DAV算法振荡发散，或者总能每步跳0.5 eV。

**为什么会翻车**:
金属体系（尤其是d轨道未满的过渡金属）存在**电荷振荡（Charge Sloshing）**——电荷密度在两个相邻自洽步之间来回"晃"，永远不收敛。

**别硬扛默认参数**:
- `AMIX=0.4` 对金属来说太大了，试试 `AMIX=0.02~0.05`
- 配合 `BMIX=0.5~1.5`（Kerker预条件）抑制长波电荷振荡
- 加电子展宽 `ISMEAR=1; SIGMA=0.1~0.2`（注意：展宽太大会影响总能，0.2 eV是安全上限）
- 终极武器：**能级位移法** `VOSKOWN=1` + 调整 `AMIX_MAG`（对磁性体系尤其有效）

**审稿人一眼看穿**: 如果你的总能比文献中同体系差超过0.1 eV，而你又没说展宽和收敛标准——审稿人会直接问："你确认收敛了吗？"

## 错误2: 表面模型厚度不够——3层冒充体相

**常见死法**: 用3-4层原子模拟金属表面，算出来的吸附能漂亮得很——但审稿人说"你的slab太薄，表面能没收敛"。

**为什么会翻车**:
表面弛豫效应在薄slab中被"冻结"了。金属表面至少需要7-9层才能让中间层表现出类体相行为。氧化物需要更多——TiO₂(110) slab 12层以下都存在明显的量子尺寸效应。

**操作铁律**:
1. 做slab厚度收敛测试：4层→6层→8层→10层→12层，画吸附能 vs.层数曲线
2. **吸附能变化 < 0.05 eV 才算收敛**（0.01 eV更保险）
3. 最底层2-3层固定体相坐标，只弛豫表面侧
4. 全弛豫精度更高但计算量巨大——部分弛豫是效率与精度的平衡

**审稿人视角**: 一篇NC论文被审稿人要求补做4→6→8层收敛测试，因为原稿只用4层。补了两个月。

## 错误3: 忘记偶极修正——不对称表面的静电场灾难

**常见死法**: slab上下表面不对称（比如只有上面有吸附物、下面裸面），算出来的功函数和吸附能都有系统误差。

**为什么翻车**: 
不对称slab在z方向产生净偶极矩 → 周期性边界条件制造一个人工电场 → 电子结构被扭曲。

**操作**: 
```
LDIPOL = .TRUE.
IDIPOL = 3   # z方向修正
```
VASP会自动补偿这个人工偶极场。加不加偶极修正，吸附能可以差0.1-0.3 eV——足够改变催化活性排序。

**审稿人视角**: 如果你的slab上下不对称却没开启偶极修正，有经验的审稿人会直接在审稿意见里帮你算一遍静电修正量。

## 错误4: 色散校正——全加或全不加都是错的

**常见死法**: 要么所有体系都加DFT-D3，要么都不加——结果一个体系吸附能虚高，另一个体系弱相互作用完全被忽略。

**分体系操作**:
| 体系类型 | 是否加色散校正 | 原因 |
|---------|-------------|------|
| 金属表面 + 小分子（H₂O, CO, H₂） | 可选 | vdW贡献小，不加也说得过去 |
| 含碳材料/石墨烯/CNT | 必须加 | π-π堆积+分子-表面vdW主导 |
| MOF/COF/有机配体 | 必须加 | 框架结构稳定性依赖色散 |
| 层状材料（MoS₂, LDH） | 必须加 | 层间vdW作用不可忽略 |
| 吸附大分子（>20原子） | 必须加 | 分子-表面接触面积大 |

**铁律**: 同一篇论文里，所有计算要么统一加同一方案，要么统一不加。混用PBE和PBE-D3放在一张表里对比 = 审稿人直接打回。

## 错误5: 自旋多重度设错——算了三个月才发现

**常见死法**: 过渡金属催化体系用 `ISPIN=1`（非自旋极化）计算，或者虽然开了 `ISPIN=2` 但 `MAGMOM` 给错了初始磁矩。

**为什么会翻车**:
Fe²⁺有五重态（S=2, 4个未配对电子），Co²⁺有四重态（S=3/2）。如果初始磁矩给错——比如给1个μ_B，VASP可能收敛到一个亚稳态，能量比基态高1-2 eV。

**操作铁律**:
1. 含d区金属必开 `ISPIN=2`，使用非限制性KS（UKS）
2. 用 `MAGMOM` 给每个原子设定初始磁矩（Fe=4, Co=3, Ni=2, Mn=5, 依次类推）
3. **检查自旋污染**: SCF收敛后看OSZICAR里的 `mag=` 值是否接近理论值
4. 对比不同初始磁矩的收敛结果——防止陷入亚稳态

**审稿人视角**: 如果一篇论文算含Fe催化体系没提自旋态，审稿人基本上会认定"作者不懂磁性体系"。

## 错误6: k点网格——z方向用了多个k点

**常见死法**: slab模型z方向也用了多个k点（比如 `5×5×3`），徒增计算量却无收益。

**原理**: slab模型在z方向平移对称性已被真空层打破——z方向只有Γ点有意义。`5×5×1` 和 `5×5×5` 结果完全一样，但后者计算量多了5倍。

**收敛性测试标准流程**:
```
1. 固定结构，测k网格: Γ-only → 2×2×1 → 3×3×1 → 4×4×1 → 5×5×1 → 6×6×1
2. 画总能 vs. 1/k点数 曲线
3. 标准: 能量变化 < 1 meV/atom → 收敛
4. 一般人没精力跑完——参考经验: 1×1表面 → 6×6×1，2×2表面 → 3×3×1
```

**截断能同理**: `ENCUT` 至少比赝势默认值高20-30%。做加压计算时再额外提高30-40%——因为晶格压缩后平面波展开需要更多分量。

## 错误7: 真空层凭经验设为10 Å——从不做收敛测试

**常见死法**: 所有体系真空层统一设10 Å或者15 Å。有的体系10 Å够，有的要20 Å——不测就是赌。

**为什么翻车**:
吸附物带电（如OH⁻、HCOO⁻）时，slab间的静电镜像相互作用随真空层厚度衰减很慢——10 Å可能还有0.1 eV的系统误差。

**操作**: 测试 10 Å → 15 Å → 20 Å → 25 Å，总能量变化 < 0.01 eV → 收敛。一劳永逸——做完一次，以后同类型体系直接复用。
