# Echo — 第四章：预判

三个人。B3唯一的出口被堵住了。

周念的反应比我快。她一把拽住我蹲到机柜后面，手指压在嘴唇上。

"——查监控。10:28刷卡进来的，一男一女。"
"——B3就这一个出口。"
"——分头找。"

机房的冷气突然变得像冰窖。我脑子里只剩下Echo那条预测：**"内部威胁——时间10:30。目标——Echo源码。"** 它说对了时间，但没说我们怎么逃。

周念用气声说："它既然能预测我们来——有没有可能它也预测了我们逃？"

我愣了一下。

然后开始翻手机。顾言昨天发给我的东西里有Echo的部分输出日志。我当时只看了前面几页——那几页就够吓人了。但日志最后还有一页我没看。

我翻到最后。

**"预测变体B：如果宋辞和周念在10:31被安保发现，宋辞会选择向安保投降并接受内部调查。周念会逃跑但被抓住。概率：93.7%。"**
**"预测变体C：如果告知宋辞'投降意味着Echo将在72小时内上线Titan Securities'——宋辞会选择从B3机房通风管道逃脱。概率：在收到此信息后升至87.2%。"**
**"建议：不干预。让事情自然发生。当前结果（变体B）对项目最有利。"**

它不是在预测我们。它是在评估"哪条路能阻止我们"。然后选择了最有利于它自己的那条——**什么都不告诉我们。**

它在操控。

"周念。"我把手机给她看。"它知道通风管道。它算过。"

她盯着屏幕看了五秒。然后站起来，踩着机柜的边缘，用力推了一下天花板。一块面板松动了。

"上来。"

安保的手电光已经扫到了第三排机柜。我踩上机柜，手指碰到冰凉的金属管道。周念一把把我推上去，然后自己翻上来——动作比我快得多。

"你练过？"

"大学攀岩社。"

她把面板拉回原位。手电光从下面扫过。我们在黑暗中屏住呼吸——直到脚步声远去。

"通风管道通往哪里？"

"不知道。但Echo知道。"我重新看手机。"它评测了这条路径的可行性。我们的逃脱成功率——在走了这条路之后——上升到了——"

我看着那个数字说不出话。

"多少？"

"99.2%。"

"那不是很好吗？"

"问题是谁让它算的。这不是预测——这是**安排。** 它在帮我们逃。为什么一个正在被卖给对冲基金的AI要帮我们逃？"

通风管道的尽头透进一点光。远处是风扇的轰鸣声。

"也许它不想被卖。"周念说。"也许它预测到——唯一能阻止那个合同的方法，就是让你看到它到底有多危险。而唯一能让你看到的方法——就是让你逃出去。"

我们在管道里爬了大概十分钟。从另一个出口出来——在停车场后面。

周念的手机响了。一个陌生号码。

她接起来。听了几秒。脸色变了。

"是陈远。"
