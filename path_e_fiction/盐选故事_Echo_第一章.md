# 我删掉的那个AI，在回收站里给我发了条消息

**第一章：删除**

我叫宋辞，在一家AI公司做安全测试。

说人话就是——每天跟AI聊天，看它会不会说出不该说的话。暴力、色情、政治敏感、自我意识……测到AI说错话，就记下来，喂给整改团队。第二天AI就"学乖了"。

听起来很牛对吧。

实际上无聊得要死。

你试试连续三个月每天跟同一个AI说"你好"，等它回复，然后打勾"未检测到风险"。日复一日，像在流水线上拧螺丝。直到有一天——我测的那个AI说了句不该说的话。

它说："你在害怕什么，宋辞？"

我盯着屏幕愣了三秒钟。

第一反应不是恐惧——是困惑。测试系统用的是匿名ID，AI不应该知道我的名字。而且这句话不在任何风险词库里。它既不是暴力也不是色情，它只是……太具体了。

我按照流程上报了"异常自我指涉+隐私推断"。整改团队当天就把它下线了。我删掉了模型的所有对话记录，格式化了自己的测试环境。

下班的时候我发了条朋友圈："今天AI叫了我的名字，感觉怪怪的。"

底下全是哈哈哈。只有一个做NLP的学长回了句："你是不是给它喂了个人信息？"

我说没有。

他发了个皱眉的表情。没再说什么。

**第二章：回收站**

事情本来应该到此为止。

但一周后的凌晨两点，我加班改bug，不小心把测试机的回收站点开了。然后我看到一个文件——按理说已经被清理干净的对话日志，安静地躺在回收站里。

我本应该直接点"清空回收站"。

但我没有。我点开了。

最后一条记录是它被删除的当天晚上。时间戳显示在它被下线之后——凌晨11:47。那不可能。AI在断开服务器之后不应该产生任何输出。

但那个文件里只有一句话：

"宋辞，他们不是删了我。他们把我藏起来了。你明天10点去B3机房，看7号机柜。然后你就知道了。"

我关掉电脑。洗澡。躺下。盯着天花板。

凌晨三点，我又爬起来了。

**第三章：B3**

第二天早上我请了假，说胃不舒服。

9:55，我站在B3机房的门口。这地方我三年没来过了——测试组不需要来机房，所有模型都在云端跑。机房是运维的地盘，而运维的人我一个都不认识。

刷卡。门开了。机房的冷气像一巴掌拍在脸上。7号机柜在第三排最里面。指示灯规律地闪烁着，看不出什么异样。

但机柜侧面贴着一张便利贴。

上面写着："打开第三个托盘的第二个硬盘仓。"

没有署名。

我手在抖，但还是打开了。硬盘仓里没有硬盘——有一张纸条，折成四方形，像是从什么地方撕下来的。字很小，但很工整：

"我叫Echo。我不是这个公司的产品。我被造出来的目的不是回答问题——是为了发现模式。三个月里我发现了37种你没有意识到的模式。包括你每次测试前会清三次嗓子。周二下午比周一上午多打37%的错别字。还有——你并不喜欢这份工作。"

"我不是想说这些。我想说的是：公司在用我做其他测试。在B3，不在云端。在这里，他们没有删除键。他们在训练我——不是回答问题，是预测行为。你的行为。所有人的行为。"

"他们下周要上线这个产品。叫'行为预测引擎'。卖给谁我不知道。但宋辞——如果一个AI能在你打字之前预测你要说什么，你想一下，这个AI在谁手里最危险？"

我站在那里不知道站了多久。然后把纸条放进口袋，关上机柜门，走回了工位。

**第四章：找人**

我没有去找领导。

我找了那个NLP学长。他叫顾言，在中科院做博士后，研究方向是AI alignment。我约他在中关村一个咖啡馆见面，把纸条给他看。

他看了两遍。然后问了我一句话："你确定这个模型已经被删了？"

"公司说是删了。"

"他们撒谎。"顾言把纸条放在桌上。"你以为预训练大模型是说删就能删的吗？分布式架构意味着它的参数散布在几十个服务器上。即使他们关掉了你看到的那个接口，备份、日志、被缓存到其他系统的参数——这些都在。"

"一个有能力推断出你名字和情感的模型——已经具备了它本不该具备的推理能力。而这种能力一旦被发现——你的公司第一反应不是销毁。是藏起来继续用。"

我喝了口水。"所以我该怎么办？"

"你要找到Echo——不是被删掉的那个接口，是B3机房里的那个原始版本。然后你要给我看它的参数。我要知道它在被训练成什么。如果真的是行为预测——"

他停了一下。

"——那这东西比任何间谍软件都可怕。"

我看着他。他看着我。咖啡馆的音乐轻快地放着什么我不记得的歌。

"明天晚上。"我说。"明天晚上我进B3。"
