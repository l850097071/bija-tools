## 陷阱1: POTIM选错了——你用0.5的步长求二阶导数，精度不如瞎猜

**死法**: IBRION=5（振动频率），不设POTIM，用默认值0.5。算出来的频率和实验差10-20%。

**物理原因**: VASP的IBRION=5用有限差分算Hessian——每个原子沿±x, ±y, ±z位移，从力变化求能量二阶导数。位移步长POTIM是关键超参数：太小→力变化淹没在数值噪声里→Hessian的60%以上分量是噪声；太大→势能面非谐性介入→二阶导数近似失效。

**量化灾难**:
```
POTIM=0.5 Å → 位移太大，非谐性贡献 → 高频模式系统偏低
POTIM=0.005 Å → 位移太小，数值噪声主导 → 低频模式随机误差>50%
POTIM=0.015 Å → 最优（DFT力精度~0.001 eV/Å时）
```

**为什么0.015**: DFT的力精度在0.001 eV/Å量级。你需要位移产生的力变化至少比噪声大一个数量级。对典型的振动模式力常数（~1-10 eV/Å²），0.015 Å的位移产生~0.015-0.15 eV/Å的力变化——恰好高出噪声一个数量级，又不至于进入非谐区。

**操作铁律**:
```
孤立分子/团簇: POTIM = 0.015  (默认0.5是给周期性体系优化的，不是分子)
表面吸附质:   POTIM = 0.010-0.020  (力精度比分子稍好)
周期性固体:   POTIM = 0.005-0.010  (因为原子数多，Hessian对角占优)
IBRION=6 (对称差分): 位移*2→力变化*2→精度更好，耗时*2
```

**审稿人怎么抓到**: 如果你报告某个振动模式和实验差>5%且没讨论POTIM收敛，审稿人可以合理质疑你的频率精度。

## 陷阱2: 选择性动力学（Selective Dynamics）冻结原子——你以为省了机时，其实毁了频率

**死法**: 计算表面吸附质的振动频率时，把底下几层衬底原子冻结（F F F），只放开吸附质原子（T T T）。频率出来了，全正，没问题——对吧？

**物理真相**: 你冻结了衬底原子→Hessian矩阵里所有衬底-吸附质耦合项被强制设为零→高频模式（牵涉衬底振动）被错误压低→低频模式（纯吸附质运动）被错误抬高（因为少了下层原子的"弹簧"贡献）。对于表面吸附能计算，这个误差在ZPE校准时表现为：你的ZPE比真实值低20-50 meV。

**量化**: 以CO在Pt(111)表面为例。冻住Pt下面两层→C-O伸缩频率误差~10 cm⁻¹（较小，因为是纯吸附质内部模式）→Pt-C伸缩模式误差~50 cm⁻¹（巨大，因为牵涉衬底Pt位移）。

**操作铁律**:
```
✅ 至少放开吸附质+第一层衬底（吸附质的"床垫"不能冻）
✅ 验证方案: 对比放开2层vs3层衬底→频率RMS变化<5 cm⁻¹→收敛
❌ 只冻底层、不验证收敛 → 审稿人可质疑所有频率
```

**反直觉**: 放开更多原子后虚频反而可能消失——因为衬底原子提供了真实的力学响应，"假紧"的Hessian变成了"真松"的Hessian。所以"冻结原子→少虚频"是危险的错觉。

## 陷阱3: Γ点振动频率和声子谱Γ点频率不一致——不是bug，是你忘了

**死法**: 你用IBRION=5算了一个2×2×2超胞在Gamma点的振动频率（有N个频率值）。又用phonopy跑了一遍同一个2×2×2超胞的声子谱。声子谱Gamma点的频率和IBRION=5的直接频率不一样——你以为phonopy算错了。

**物理原因**: 这不是bug。IBRION=5算的是你2×2×2超胞的Gamma点振动模式（q=0，所有原子同相位振动）。phonopy Gamma点频率是通过力常数Fourier插值得到的——它用的是实空间超胞力常数矩阵插值回Gamma点。两个数值应该接近但不完全相等，差异来自phonopy的Fourier插值误差。

**但是**：差异>5 cm⁻¹ → 你的超胞不够大（phonopy力常数截断误差显著）。差异>20 cm⁻¹ → 必须增大超胞。

**操作铁律**:
```
IBRION=5 Gamma点频率 = 真值基准（直接用超胞算的，无插值误差）
phonopy Gamma点频率 = Fourier插值 → 与基准的偏差 ≈ 力常数截断误差
偏差<3 cm⁻¹ → 超胞够大。偏差>10 cm⁻¹ → 增超胞。
```

**一个审稿人特喜欢问的刁钻问题**: "You computed both the Gamma-point frequencies and the phonon dispersion. Why are the Gamma-point frequencies slightly different between the two methods?" 如果你答不出插值误差→审稿人判定你"对方法理解不深"。

## 陷阱4: 零点能校正不是"做了就对了"——它会放大你的能量误差

**死法**: 算完吸附能→加ZPE校正→报告"经零点能校正的吸附能为X"。审稿人问你ZPE校正的不确定度→你答不出来。

**物理原因**: ZPE = ½ Σ hν_i。你的频率每个都有误差δν_i。ZPE的不确定度 = ½ h Σ δν_i。如果20个振动模式每个误差±10 cm⁻¹ = ±1.24 meV/模式 → ZPE总不确定度~±25 meV（累积了20个模式）。而你的吸附能才100-500 meV——你的ZPE校正本身带着±25 meV的不确定度。

更要命的是：对于反应能ΔE = E(products) - E(reactants)，反应物和产物各有一个ZPE，误差不抵消——它们**叠加**: ΔZPE不确定性 = sqrt( σ²(reactants) + σ²(products) )。如果你的反应物和产物频率精度都是15 cm⁻¹，ΔZPE总不确定度可能到30-50 meV——在某些反应里，这已经比反应能本身的变化大了。

**操作铁律**:
```
强制报告: "ZPE校正 = X eV, 估计不确定度 = ±Y eV (基于频率精度δν=__ cm⁻¹)"
如果 Y > 0.3×|ΔE| → ZPE校正无意义，讨论其局限性
```

**安全区间**: 
- 频率精度 < 5 cm⁻¹ → ZPE校正可靠（孤立分子/bulk）
- 频率精度 5-15 cm⁻¹ → ZPE校正定性可用（表面吸附质常见区间）
- 频率精度 > 20 cm⁻¹ → ZPE校正仅参考（大超胞/低频模式多）

## 陷阱5: 过渡态验证——一个虚频不够，得是对的那个虚频

**死法**: NEB/IDM找到过渡态→跑IBRION=5→输出1个虚频→开心写"transition state confirmed with exactly one imaginary frequency"。

**物理真相**: 过渡态的虚频必须是**沿着反应坐标的振动模式**。你不能只看有几个虚频——你必须看虚频对应的振动模式是不是真的在"反应物→产物"的方向上。有且仅有一个虚频，但如果这个虚频是甲基旋转或者水分子晃动——这不是过渡态，这是其他自由度的鞍点。

**操作铁律**:
```
过渡态验证三步检查:
1. 有且仅有一个虚频 ✓ (必要条件，不充分)
2. 虚频的振动本征矢 = 沿反应坐标方向 → 肉眼观察或计算投影
   用VASPKIT的 508 功能或手动提取 eigenvector → 动画核对
3. 将TS沿虚频本征矢方向位移±0.1 Å → 做两个端点优化 → 两端分别落到反应物和产物
   如果某一端没落到预期的极小点 → TS不对或反应路径不对
```

**审稿人怎么抓到**: 报告了TS的虚频值但不展示虚频对应的振动模式。审稿人合理要求补充→如果你展示的振动方向和宣称的反应坐标不一致→直接拒稿。

## 陷阱6: 软模式的2D/层状材料陷阱——你的超胞看似够，实则不够

**死法**: 算2D材料（MoS₂, graphene, g-C₃N₄等）的声子谱。3×3×1超胞，没虚频，声学支色散正常——你以为对了。

**物理真相**: 2D材料的ZA（面外声学）模式色散关系在长波极限下是 ω ∝ q²（不是线性 ω ∝ q）。这是因为2D材料没有z方向平移对称性，面外弯曲不消耗能量（零厚度→零弯曲刚度）。这个q²色散意味着长波ZA模式极其软——在超胞不够大时，Gamma点附近的ZA模式被Fourier插值错误地"拉直"为线性色散。

**后果**: 声子谱的"Gamma点近邻"区域（q→0）ZA模式频率被高估。这直接影响：
- Debye温度和比热容的低温行为（错误预测T³定律的系数）
- 晶格热导率（LA/TA/ZA模式竞争→错估ZA贡献）
- 声子辅助的光学过程

**操作铁律**:
```
2D材料声子谱 → 至少4×4×1超胞。如有软模式→5×5×1甚至6×6×1
检查: Gamma点近邻(0.05,0,0)处ZA模式频率 < 0.1 THz → 超胞可能不够
Gold标准: N×N×1和(N+1)×(N+1)×1超胞的ZA色散形状一致
```

## 陷阱7: 对称性降级后Hessian的"幽灵虚频"

**死法**: 你算了一个高对称性结构（比如C₃v对称的NH₃在Pt(111)上）的振动频率→没虚频。然后你想测试更低对称性的吸附构型→移开吸附质0.1 Å→弛豫→跑IBRION=5→出来两个虚频→你想"结构不稳定"→换初始结构重算。

**物理真相**: 对称性降低后的虚频可能是Hessian矩阵的数值病态——不是因为结构不稳定，而是因为你的POTIM和力精度在低对称性下不足。高对称性时，许多Hessian矩阵元被对称性约束为零→矩阵条件数好→对角化稳定。对称性打破后，所有矩阵元都可能有非零值→Hessian条件数恶化→数值对角化产生虚假负本征值。

**操作铁律**:
```
低对称性振动频率→虚频判别比高对称性放宽3倍:
  高对称性: |虚频| > 5 cm⁻¹ → 可能真虚频
  低对称性: |虚频| > 15 cm⁻¹ 才可能是真虚频
  原因: 低对称性Hessian的本征系统误差约10-15 cm⁻¹
POTIM设到0.01-0.02（低对称性Hessian对角元更小，需要更大的信噪比）
```

## 陷阱8: 杂化泛函的频率——你算了更贵的，结果不一定更准

**死法**: "PBE频率不准→换HSE06→肯定对"。

**物理真相**: HSE06改进了电子结构（带隙、电荷局域化），但对振动频率的改进高度依赖体系：
- **共价晶体/分子**: HSE06频率确实通常比PBE更接近实验（键长更准→力常数更准）
- **离子晶体/金属**: HSE06频率不一定比PBE好——力常数对交换关联泛函的敏感度远低于结合能
- **vdW体系**: HSE06完全没解决色散问题→频率精度≈PBE

**量化数据**（文献共识）:
```
分子振动频率: PBE ~RMS 35 cm⁻¹, HSE06 ~RMS 25 cm⁻¹ → HSE改善~30%
固体声子频率: PBE ~RMS 15 cm⁻¹, HSE06 ~RMS 13 cm⁻¹ → HSE改善~13% (不显著)
```

且HSE06做振动频率的计算成本是PBE的10-50倍。审稿人如果看到你用HSE06做振动频率→会合理期待你讨论"为什么需要用HSE而不是PBE"。

**操作铁律**:
```
只需PBE: 纯频率检查（有无虚频）、定性趋势、反应能ZPE校正（误差抵消）
需要HSE: 与实验频率做定量对比、IR/Raman强度计算、高精度热力学
不可接受: 用HSE06但不解释为什么PBE不够→审稿人质疑方法论合理性
```

## 陷阱9: DFPT（密度泛函微扰）的K点陷阱——你以为"密度"是对的，其实是错的

**死法**: 用DFPT（IBRION=7/8, LEPSILON=.TRUE.）算声子色散→色散曲线光滑漂亮→审稿人问你"你的q点网格收敛了吗"→你沉默。

**物理原因**: DFPT直接在倒空间计算动力学矩阵D(q)，避免实空间超胞→避免超胞截断误差——这听起来完美。但DFPT需要你为每个q点单独跑一遍SCF计算。如果你的q点网格和SCF K点网格不匹配→D(q)矩阵质量差→Fourier插值到精细q点网格→产生虚假波纹。

**收敛测试（大多数人不做）**:
```
Step 1: q点3×3×3 → 计算声子色散
Step 2: q点4×4×4 → 计算声子色散  
Step 3: q点5×5×5 → 计算声子色散
三者高频光学支偏差 < 0.2 THz → q点收敛
SCF K点密度 = q点密度 × 至少4 → 确保每个q点的SCF精度一致
```

**最容易错的体系**: 金属（需要密K点→DFPT每个q点的SCF都贵）和极性半导体（长程库仑→需要非常密的q点才能收敛LO-TO劈裂）。

## 陷阱10: "力常数截断半径"——phonopy给你的默认值可能害了你

**死法**: 用phonopy默认设置，超胞3×3×3→生成12×12×12 q点声子色散→看到完整色散→提交。

**物理真相**: phonopy默认力常数截断半径 = 第N近邻距离（基于超胞尺寸自动估算）。如果默认截断半径太大→包含了很多数值噪声（远距离力常数实际为零，但噪声被Fourier插值放大）。如果默认截断半径太小→丢失了真实的长程力常数→色散形状错误。

**对于极性材料（有Born有效电荷→长程q→0的|q|发散）**：你的力常数需要在q→0时处理LO-TO劈裂。phonopy的`--nac`选项需要Born有效电荷+介电常数。如果你没算这些或者算错了→LO-TO劈裂的位置和大小全错→声子色散在Gamma点附近的形状不可信。

**操作铁律**:
```
phonopy设置检查单:
[ ] 力常数截断半径 → 检查 `FC_*` 文件中最远未截断的距离 → 不含噪声
[ ] 非解析项校正 → `--nac` + BORN文件 + 介电常数 → 极性材料必做
[ ] q点网格 → 至少32×32×32（用于色散曲线插值）→ 也做收敛测试
[ ] 验证: phonopy Gamma点频率 vs 有限差分Gamma点频率 → < 3 cm⁻¹
```

