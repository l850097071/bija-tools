# 避坑#23: DFT范德华修正——D3/D4/TS/MBD怎么选才不会翻车？

**避坑系列第23篇 | 知乎笔名发布 | 约4200字**

---

计算化学里有一个尴尬的事实：90%的人用范德华修正的方式是"别人用D3我就用D3"。至于为什么不是D2、D4、TS、MBD——很少有人能说清楚。

这篇把5种主流vdW修正的差异、陷阱、选择逻辑一次讲清。尤其第4个陷阱（D3对过渡金属的Becke-Johnson阻尼）和第6个（MBD对层状材料的必须性），踩过的人不在少数。

---

## 一、先搞清楚：范德华修正在修正什么？

标准GGA泛函（PBE、PBEsol、RPBE）对色散力的描述是**零**——长程1/R⁶衰减的瞬时偶极-诱导偶极相互作用完全不包含在内。这对以下体系是灾难性的：

- 层状材料（石墨、MoS₂、h-BN）→ 层间距差0.5-1.0 Å
- 分子晶体（冰、多环芳烃）→ 晶格参数严重偏离
- 表面吸附（分子/催化剂表面）→ 吸附能差0.3-0.5 eV
- 弱结合复合物（惰性气体二聚体、主客体）→ 结合能可能变号

**vdW修正的本质**：在DFT能量上叠加一个色散校正项 E_disp = E_DFT + E_vdW。

不同的修正方法，区别在于**怎么算这个E_vdW**。

---

## 二、5种主流方法的技术差异

### 1. DFT-D2 (Grimme, 2006)
```
E_disp = -s₆ · Σ C₆ⁱʲ / R⁶ · f_damp(R)
```
- C₆系数：查表（固定值，不依赖化学环境）
- 阻尼函数：Fermi型
- **致命缺陷**：C₆不随配位环境变化。Fe原子在Fe₂O₃里和Fe(CO)₅里用同一套C₆——明显不对。

**结论**：2026年不该再用D2。期刊审稿人看到D2会要求重算。

### 2. DFT-D3 (Grimme et al., 2010) — 当前最主流
```
E_disp = -Σ Σ Cₙⁱʲ / Rⁿ · f_damp(R)  (n=6,8)
```
- C₆系数：基于原子坐标和配位数**动态计算**——解决了D2的环境依赖问题
- 两体C₈项包含在内
- 两种阻尼：**零阻尼**（original）和 **Becke-Johnson阻尼**（BJ）

**D3(BJ)是当前最安全的默认选择**——但有两个隐蔽陷阱（见下文）。

### 3. DFT-D4 (Grimme et al., 2017/2020)
- 在D3基础上引入**原子电荷依赖**——C₆系数随氧化态变化
- 适用于变价金属体系（Mn²⁺ vs Mn⁴⁺, Fe²⁺ vs Fe³⁺等）
- **代价**：比D3慢约10-20%，且不是所有主流代码都原生支持

**何时升级到D4**：体系含变价过渡金属，且氧化态对结果敏感性高。

### 4. TS (Tkatchenko-Scheffler, 2009-2012)
```
C₆_eff = (V_eff / V_free)² · C₆_free
```
- C₆通过Hirshfeld体积比从自由原子值缩放——考虑了化学环境
- 不需要元素参数——全电子密度驱动
- **优势**：对新型材料（无经验参数覆盖的元素组合）更物理

**陷阱**：TS对电子密度划分（Hirshfeld partitioning）敏感，不同代码实现可能有差异。

### 5. MBD (Many-Body Dispersion, Tkatchenko et al., 2012-2017)
- 在TS基础上加入**多体色散耦合**——通过耦合谐振子模型
- 描述超越两体加和的色散效应——对**极化率各向异性**大的体系重要
- **计算开销**：比D3大10-100倍

**何时必须用MBD**：层状材料层间结合能、分子晶体晶格能、纳米结构自组装。

## 三、7个隐蔽陷阱（核心避坑内容）

### 陷阱1：D2的C₆环境不敏感——审稿拒稿雷区
**表现**：含过渡金属的吸附体系用D2，审稿人一句"why not D3 or D4?"就可以打回来。
**根因**：D2的C₆系数是元素级别的固定值，Fe原子不论在什么化学环境都用同一套——这在2026年标准下是不可接受的。
**正确做法**：至少用D3(BJ)。含变价金属→考虑D4。

### 陷阱2：D3零阻尼 vs BJ阻尼——选错差0.3 eV
**表现**：同一个体系，D3(zero)和D3(BJ)的吸附能差0.2-0.4 eV——比很多"化学效应"都大。
**根因**：零阻尼在中等距离（~3-5 Å）阻尼过快，低估了中程色散。BJ阻尼的衰减更平滑，更接近精确的范德华C₆/R⁶渐近行为。
**正确做法**：除非你很清楚为什么用零阻尼，否则一律用BJ。VASP中 `IVDW=11` 是D3(zero)，`IVDW=12` 是D3(BJ)。**多数人不知道这个区别就选了默认的11——翻车重灾区**。

### 陷阱3：D3对3d过渡金属的C₆参数——至今不完整
**表现**：部分3d金属（尤其Cr、Mn、Co的低氧化态）的D3参数是从有限数据外推的。
**根因**：Grimme原始参数化时，某些过渡金属的参考数据稀疏。实际测试中，Mn²⁺体系的D3吸附能误差可达0.3-0.5 eV。
**正确做法**：含3d金属的体系——用D4替代D3，或至少用D3(BJ)+MBD交叉验证关键结论。

### 陷阱4：D3(BJ)对碱金属/碱土金属——参数边界外推
**表现**：Na、K、Ca等体系用D3，C₆可能被高估。
**根因**：D3参数化基于中性主族分子的训练集——碱金属在训练集中代表性不足。
**正确做法**：碱金属体系优先用TS或MBD（电子密度驱动，不依赖原子参数表）。

### 陷阱5：TS/MBD的电子密度收敛要求
**表现**：用TS/MBD，换一个泛函（PBE→PBEsol），吸附能从-0.5变成-0.2 eV——不是化学变了，是Hirshfeld体积变了。
**根因**：TS的C₆缩放依赖Hirshfeld体积比(V_eff/V_free)。电子密度的小差异→体积比差异→C₆差异放大。PBE和PBEsol的电子密度分布不同，导致TS校正量不同。
**正确做法**：用TS/MBD时，报告电子密度泛函+vdW方法组合，不要单独换泛函。审稿人会问"你换泛函后vdW方法是否一致？"

### 陷阱6：层状材料用D3代替MBD——层间距偏0.3-0.5 Å
**表现**：石墨层间距实验值3.35 Å，PBE+D3=3.45-3.55 Å，PBE+MBD=3.30-3.40 Å。
**根因**：多体色散耦合（many-body dispersion）在层状材料中贡献显著——相邻层间的极化涨落相互增强。两体D3/D4完全捕捉不到这种多体协同。
**正确做法**：层状材料/2D异质结/分子晶体→**必须用MBD**。层间结合能差可达50-100 meV/原子——在相稳定性判断上就是生与死的区别。

### 陷阱7：VASP中IVDW和vdW-DF混淆——不是一回事
**表现**：很多新手以为"vdW修正"就是选一个IVDW标签。实际上VASP有三套完全不同的vdW方案。
**根因**：
- `IVDW=1/10/11/12`：DFT-D2/D3系列——在能量上加色散校正项
- `IVDW=2/20/21/202`：vdW-DF系列——在泛函本身的非局域关联核里嵌入色散
- `IVDW=3/4`：TS/MBD系列——电子密度驱动

这三套方案的物理基础和适用场景完全不同，不是"参数不一样"的问题。
**正确做法**：论文方法部分写清楚 "PBE-D3(BJ)" 或 "optB88-vdW" 而不是模糊的 "DFT+vdW"。

## 四、选择决策树（一张图搞定）

```
你的体系是？
│
├─ 含层状结构/分子晶体/2D异质结？
│   └─ YES → MBD（不可替代）
│
├─ 含3d/4d过渡金属？
│   ├─ 变价金属(Mn/Fe/Cr不同氧化态) → D4
│   └─ 稳定氧化态 → D3(BJ) + MBD交叉验证
│
├─ 含碱金属/碱土金属(Na/K/Ca/Mg)？
│   └─ TS或MBD（D3参数不可靠）
│
├─ 主族元素分子/常规表面吸附？
│   └─ D3(BJ) ← 当前最安全默认选择
│
├─ 需要极高精度(+有机分子构象能)？
│   └─ D4 + MBD双验证
│
└─ 审稿人要求"state-of-the-art"？
    └─ MBD（无可争议的当代最佳）
```

## 六、自查清单

在一个DFT项目开始前，问自己5个问题：

1. **我的体系里有层状结构吗？** 是→MBD，不是→下一问
2. **我的体系里有过渡金属吗？** 是→D3(BJ)至少，变价→D4。不是→下一问
3. **我用的IVDW标签是几？** 11→考虑改12；其他→确认理解含义
4. **我论文method部分写了具体的vdW方法名吗？** 没有→加上 "PBE-D3(BJ)" 而不是 "DFT+vdW"
5. **我的关键结论对vdW方法敏感吗？** 没测过→用至少两种vdW方法交叉验证

## 七、总结（给直接拉到底的人）

- **停止使用D2**。2026年任何期刊审稿人都不会接受。
- **D3(BJ)是当前最安全的默认选择**——但VASP中IVDW=12，不是11。
- **层状材料/2D材料/分子晶体 → 必须用MBD**。两体校正不够。
- **含变价过渡金属 → 升级到D4**。
- **碱金属体系 → 用TS/MBD替代D3**。
- **论文写清楚"PBE-D3(BJ)"，不要写模糊的"DFT+vdW"**。
- **关键结论用至少两种vdW方法交叉验证**。
