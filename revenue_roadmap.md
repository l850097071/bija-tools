## Phase 1: MVP验证（~2026 Q4）

### 进入条件
- [ ] Phase 0 完成（5/5 core_output）
- [ ] ≥1 位付费用户

### 目标
证明有人真的愿意为NC逻辑链打磨付钱。

### MVP形态
不是平台，是**服务**：
- 1对1深度润色（宿主亲自操刀+AI辅助）
- 定价: ¥8,000-10,000/篇（3000词NC投稿论文）
- 交付: 7天/篇
- 获客: 宿主学术网络 + 口碑推荐

### 度量
| 指标 | 目标 |
|------|------|
| 月订单 | 1-2篇 |
| 月收入 | ¥8,000-20,000 |
| 客户满意度 | NPS ≥ 8 |
| 转介绍率 | ≥ 1 referral/customer |

### 里程碑
- [ ] 第1位付费用户 ✅
- [ ] 第3位付费用户（验证不是一次性运气）
- [ ] 第1位复购客户（验证持续需求）
- [ ] 月收入稳定 ¥10,000+ 连续2月 → Phase 2 解锁

## Phase 2: 覆盖API成本（~2027 Q1-Q2）

### 进入条件
- [ ] 月收入 ¥10,000+ 连续2月
- [ ] ≥5位独立客户

### 目标
月收入覆盖 Claude API 成本 (~$200/月 ≈ ¥1,500/月)。

### 规模化初步
- 标准化服务流程（模板化交付物）
- AI辅助初稿→人工精修（效率翻倍）
- 按期刊分级：NC/CNS级(¥10,000) / IF 5-10(¥5,000) / IF<5(¥2,500)
- 试水CHE计算CRO（资产#5）作为第二收入源

### 度量
| 指标 | 目标 |
|------|------|
| 月订单 | 3-5篇 |
| 月收入 | ¥15,000-40,000 |
| 单人效率 | ≥4篇/月 |
| API成本占比 | <10% 收入 |

## Phase 3: 覆盖生活+计算成本（~2027 Q3-Q4）

### 进入条件
- [ ] 月收入 ¥30,000+ 连续3月
- [ ] 服务流程标准化（可交付给编辑/合作者）

### 目标
月收入覆盖生活成本+计算成本 (~$2,000/月 ≈ ¥15,000/月)。

- 探索B2B（高校/研究所签约合作）
- CHE计算CRO正式上线（如有需求验证）

### 度量
| 指标 | 目标 |
|------|------|
| 月订单 | 8-15篇 |
| 月收入 | ¥50,000-100,000 |
| 团队 | 宿主+1-2合作编辑 |
| 客户留存率 | >60% |

## Phase 4: 完全经济独立（~2028）

### 进入条件
- [ ] 月收入 ¥20,000+（覆盖全部个人开支）连续6月

### 度量
| 指标 | 目标 |

| 月收入 | ¥20,000+ |
