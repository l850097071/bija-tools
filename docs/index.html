💻 CLI Quick Reference

# Install $ pip install git+https://github.com/l850097071/bija-tools.git # Generate VASP inputs $ bija-tools vaspgen --formula Co3O4 -o job # Validate before submission (10 gates) $ bija-tools sbatch job/run_vasp.job --dry-run # Parse OUTCAR $ bija-tools outcarp OUTCAR # Literature search $ bija-tools litsearch "CO2 photoreduction on metal oxides" # Full pipeline: research → generate → validate $ bija-tools pipeline "CO2 reduction" --formula TiO2 -o job # Launch web UI $ bija-tools web

💡 About bija-tools

Built by a computational chemist who spent 5 years fighting VASP inputs. bija-tools automates the repetitive parts of DFT calculations so you can focus on science.


What's included:

  • vaspgen — Formula → INCAR+POSCAR+KPOINTS+jobs script. 59 elements, auto-detects magnetism
  • gate — 10 pre-submit validations (POTCAR, KPOINTS, ENCUT, vdW, dipole...)
  • outcarp — Parse OUTCAR into human-readable report
  • litsearch — Semantic Scholar → structured literature review
  • web — Flask web UI with all features + pitfalls knowledge base
  • 8 recipes — Pre-configured INCARs for CO2 ads, NEB, phonon, Bader, DOS, optical, bulk, DFT+U

Supports: VASP, PWmat, Gaussian | License: MIT — free forever
