🔧 What It Does

Every feature is born from real DFT calculation pain. No bloat. Just what you actually need.

⚙️

VASP Input Generator

Formula → INCAR + POSCAR + KPOINTS + job script. Auto-detects magnetism, sets ISPIN/MAGMOM. Supports 59 elements.

bija-tools vaspgen --formula Co3O4 -o job

🛡️

Pre-Submit Gate Validator

10 automated checks before SLURM submission. POTCAR mismatch? KPOINTS too sparse? Missing vdW? Blocked before wasting GPU hours.

bija-tools sbatch run.job

📊

OUTCAR Parser & Reporter

Convergence status, energies, forces, timing → human-readable report. Batch mode watches directories for completed jobs.

bija-tools outcarp OUTCAR

🔍

Literature Deep Search

Semantic Scholar API → structured review with thematic grouping. Stay on top of the literature without leaving the terminal.

bija-tools litsearch "CO2 reduction"

🌐

Web UI (NEW)

Flask-powered web interface: VASP input generator + gate validator + pitfalls knowledge base. Point, click, generate.

bija-tools web --port 8765

📋

8 Calculation Recipes

Pre-configured INCAR templates for CO₂ adsorption, NEB, phonon, Bader, DOS, optical, bulk, DFT+U. Each with pitfall checklist.

bija-tools pipeline "CO2 reduction" -o job

⚡ Quick Start

# Install $ pip install git+https://github.com/l850097071/bija-tools.git # Generate VASP inputs for Co₃O₄ $ bija-tools vaspgen --formula Co3O4 -o my_calc # Validate before submission $ bija-tools sbatch my_calc/run_vasp.job --dry-run # Parse results $ bija-tools outcarp my_calc/OUTCAR # Full pipeline: literature → inputs → validate $ bija-tools pipeline "CO2 photoreduction" --formula TiO2 -o job # Launch web UI $ bija-tools web
