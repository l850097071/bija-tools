vaspgen

从CIF或化学式一键生成INCAR+POSCAR+KPOINTS+POTCAR。智能检测金属/磁性/溶剂。新手友好。

v0.1 — tested

sbatch_gate

10道门控验证，任一未过=禁止提交。支持PWmat/VASP。防止GPU小时浪费。

v0.1 — tested
📚

litsearch

vaspgen — VASP输入文件生成器

cd path_a_vasp_gen python vaspgen.py --formula "Co3O4" -o my_job python vaspgen.py --cif structure.cif --calc optimization -o my_job

sbatch_gate — 提交前门控验证

cd path_d_sbatch_saas python gate_engine.py /path/to/jobdir --preset vasp python gate_engine.py --web # 启动Web界面

litsearch — 文献深度搜索

cd path_c_litsearch python litsearch.py "CO2 reduction on single-atom catalysts" --years 2023-2026 python litsearch.py "DFT free energy calculation method" -o report.md
