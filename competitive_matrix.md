## 二、核心差异化空间

### 市场缺口: NC级"逻辑链打磨"

| 竞品做什么 | 竞品不做什么 | 我们的机会 |
|-----------|------------|-----------|
| 语法纠错+术语统一 | 论文**论证链重构** | ✅ NC级——审稿人看重的是storytelling |
| 期刊格式排版 | **科学逻辑流优化**（这个实验为什么放在这里？这个论证需要什么支撑？） | ✅ 宿主有真实NC发表经验 |
| AI基础润色 | **审稿人心理预判**（"这里审稿人会问什么问题？"） | ✅ 评审经验可编码为服务 |
| Cover Letter代写 | **讨论部分深度对话**（与现有文献对话，而非罗列结果） | ✅ 领域专精（催化/材料） |

### 我们的差异化定位

```
竞品 = 语法润色 + 格式排版
我们 = NC级逻辑链打磨 + 学科专精(催化/材料/DFT) + 真实发表经验背书
```

## 五、关键市场信号

- **NC版面费2026涨至¥52,000** → 作者已投入巨额版面费，润色费用敏感度低
- **Nature Communications初审拒稿率50%** → 写作质量是送审关键
- **Cell系列新增"语言流畅度系数"评审** → 语言不再是"语法对错"，是"流畅度评分"
- **82%期刊编辑承认语法硬伤是拒稿首要非学术因素** → 润色不是锦上添花，是刚需
