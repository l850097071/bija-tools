## 策略1: 双均线趋势 (最稳入门)

```
你是一个专业的量化交易员。请编写一个A股双均线策略:

标的: 沪深300ETF (510300)
逻辑: 5日均线上穿20日均线→全仓买入; 下穿→全仓卖出
时间: 2020-01-01至2025-12-31, 日频回测
风控: 单笔止损-3%, 持仓时若当日跌幅>5%强制清仓
成本: 佣金万2.5, 印花税千一(卖出), 滑点0.01%

输出: 
1. 完整Python代码 (用akshare取数据+pandas计算+matplotlib画图)
2. 回测结果: 年化收益/最大回撤/夏普比率/胜率/交易次数
3. 代码带中文注释, 每个函数不超过30行
```

```

## 策略3: ETF动量轮动 (自动切换)

```

标的池: [510300沪深300ETF, 159915创业板ETF, 510500中证500ETF, 511880银华日利]

输出:
