## 一、仓位管理铁律

### 1.1 单票上限
```
单票最大仓位 ≤ 总资金 × 15%
行业最大仓位 ≤ 总资金 × 30%
总仓位 ≤ 总资金 × 80%（永不满仓）
```

### 1.2 凯利公式简化版
```
f = (bp - q) / b
b = 盈亏比（平均盈利/平均亏损）
p = 胜率
q = 1 - p

简化规则: f* ≤ 10%（单次下注不超过总资金10%）
```

### 1.3 金字塔加仓
```
初始建仓: 目标仓位的30%
确认方向: +30%（价格朝预期方向移动≥2%）
趋势加速: +40%（突破关键位+放量确认）

绝不: 亏损加仓（摊平成本=放大错误）
```

## 二、风控体系

### 2.1 三层止损

| 层级 | 触发条件 | 动作 |
|------|---------|------|
| **硬止损** | 单票亏损 ≥ -8% | 无条件平仓 |
| **时间止损** | 持仓5日仍亏损 | 减半仓 |
| **波动止损** | 单日振幅 > 2× ATR(14) | 减1/3仓 |

### 2.2 回撤控制
```
日回撤 ≥ 3% → 当日停止新开仓
周回撤 ≥ 5% → 减仓至50%以下
月回撤 ≥ 10% → 清仓→复盘→休息1周
```

### 2.3 黑天鹅防护
```
极端事件触发（单日大盘跌>5%）:
1. 所有仓位设市价止损
2. 不抄底
3. 等待波动率回归（VIX<20）再入场
```

## 三、信号分级体系

### 3.1 四层信号权重

| 层级 | 信号类型 | 权重 | 示例 |
|------|---------|------|------|
| **S级** | 多周期+多指标共振 | 40% | 周线MACD金叉+日线布林下轨+RSI<30 |
| **A级** | 双指标共振 | 25% | EMA金叉+成交量放大 |
| **B级** | 单指标触发 | 15% | 仅RSI超卖 |
| **C级** | 情绪/新闻驱动 | 10% | 行业利好+资金流入 |

### 3.2 开仓条件
```
S级信号: 建目标仓位100%
A级信号: 建目标仓位60%
B级信号: 观察，不交易
C级信号: 观察，不交易（情绪=噪音）

最低开仓要求: A级及以上
```

## 四、混合AI系统框架

基于论文(EMA+MACD+RSI+布林带+FinBERT+XGBoost, 135.49%回报/24月):

### 4.1 技术层（量化信号）
```
EMA(12/26): 趋势方向
MACD(12,26,9): 动量确认
RSI(14): 超买超卖（<30超卖, >70超买）
布林带(20,2): 波动率边界
ATR(14): 止损距离计算
```

### 4.2 情绪层（NLP信号）
```
FinBERT: 金融新闻情绪评分
关键阈值:
  >0.3: 看多
  -0.3 ~ 0.3: 中性
  <-0.3: 看空
```

### 4.3 融合规则
```
技术层得分 + 情绪层得分 = 综合信号

技术层:
  EMA多头 +1, MACD金叉 +1, RSI<30 +1, 价格<布林下轨 +1
  max = 4

情绪层:
  FinBERT>0.3 +1, 新闻量>均值 +1
  max = 2

综合≥4 → S级 | ≥3 → A级 | <3 → 不交易
```

## 五、实盘执行规则

### 5.1 时段规则
```
开盘30分钟: 不交易（消化隔夜信息）
10:00-11:30: 主交易窗口
13:00-14:30: 次交易窗口
14:30-15:00: 只平仓不开仓（避免尾盘异动）
```

### 5.2 持仓管理
```
盈利仓位:
  浮盈≥5% → 止损移至成本价
  浮盈≥10% → 止盈1/3，止损移至+5%
  浮盈≥20% → 止盈1/2，剩余追踪止盈(ATR×3)

亏损仓位:
  浮亏≥-3% → 检查信号是否失效
  浮亏≥-5% → 减半仓
  浮亏≥-8% → 全平
```

### 5.3 每日清单
```
[ ] 大盘趋势判断（上证/创业板日线方向）
[ ] 持仓股信号更新（技术层+情绪层）
[ ] 候选池筛选（今日满足A级以上条件）
[ ] 止损位更新（ATR每日重算）
[ ] 交易日志（每笔交易记录入场理由+结果）
```

## 七、生存法则

1. **保本第一**: 不亏 > 少赚 > 多赚
2. **承认错误**: 亏损时第一反应是"我错了"而非"市场错了"
3. **不追高**: 错过 > 做错。机会永远有
4. **人在回路**: AI信号是参考，最终决策须人工确认
5. **记录一切**: 每笔交易记入场理由→事后对照→迭代信号体系
