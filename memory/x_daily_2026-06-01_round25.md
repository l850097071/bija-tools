## X61: DFT振动频率/声子谱计算常见错误

来源: WhiteRose 2025(egg-box效应) + VASP/Phonopy社区最佳实践 + 内部28/630计算经验

### 6大错误源
1. **Egg-box效应** → FFT网格能量波动→虚假虚频。实空间格点不足时，原子微小位移导致能量出现周期性波动，力常数矩阵被污染。
   - 修复: PREC=Accurate / NGX,NGY,NGZ≥2×默认 / 或增加ENCUT 30%
2. **结构未充分弛豫** → 离子位置未达势能面极小值→Gamma点虚频。残余力>0.01 eV/Å时声子谱不可靠。
   - 修复: EDIFFG=-0.005(非-0.01), NSW≥200, 对称性关闭(ISYM=0)弛豫后检查IBRION=5/6力常数
3. **超胞尺寸不足** → 周期性镜像间虚相互作用→声子色散曲线非物理展宽/分裂。2×2×2在层状/低维体系中严重不足。
   - 修复: ≥10 Å真空层(低维), 超胞每个方向≥10 Å。验证: 扩大超胞后频率变化<5 cm⁻¹
4. **LO-TO分裂遗漏** → 极性材料(离子晶体)长程库仑场→Gamma点LO-TO分裂。Born有效电荷+介电常数未计算→LO模系统性偏低。
   - 修复: LEPSILON=.TRUE. Born有效电荷+静态介电常数→Phonopy LOTO标签
5. **声学支求和规则(ASR)违反** → 数值噪声/不完备力常数→Gamma点三支声学模非零频率。
   - 修复: Phonopy --asr选项(simple/crystal)。ASR强制后检查声学支<1 cm⁻¹
6. **热力学量推导链错误** → 声子DOS→Cv/S/F(T)依赖Q点网格密度和能量截断。谐波近似在T>Debye温度1/3时失效。
   - 修复: Q网格≥2×超胞k点。非谐效应(准谐近似QHA)需Vol(P)扫描。Thermo输出验证: 0K时Cv→0, S→0

### 核心启示
振动计算链: 弛豫→力常数→声子色散→态密度→热力学量，每步精度依赖上一步。3次虚频以上→回到弛豫。
